自回归过程

考虑$x_t$是由$x_{t-1}$决定，即$x_t = a_1x_{t-1}$。假设
$$x_t = \rho x_{t-1} + \epsilon_t$$
其中，随机干扰项$\epsilon_t \sim N(0, \sigma^2)$。
序列的稳定性由决定$\rho$，波动程度由$\sigma^2$决定。


一般地，考虑序列值$x_t$可由前p个时刻的序列值表示，即
$$X_t = \rho_1 X_{t-1} + \rho_2 X_{t-2} + \rho_k X_{t-k} + \epsilon_t$$
其中，$\epsilon_t \sim N(0, \sigma^2)$；是一个p阶自回归模型AR(P)，$\rho = (\rho_0, \rho_1,…, \rho_p)^T$ 是自回归系数。

一阶自回归过程AR(1)

一阶自回归表达式：
$$X_t = \rho X_{t-1} + \epsilon_t (1)$$
其中，$\epsilon_t \sim N(0, \sigma^2)$。
求方差得：
$$\gamma_0 = \rho^2\gamma_0 + \sigma_e^2$$
$$\gamma_0 = \dfrac{\sigma_e^2}{1-\rho^2}$$
根据上得出：$\rho^2\leq 1$即$|\rho|\leq 1$。
对(1)两边同时乘以$X_{t-k}$，并求期望得：
$$E(X_{t-k}X_t) = \rho E(X_{t-k}X_{t-1}) + E(\epsilon_t X_{t-k})$$
即：
$$\gamma_k = \rho \gamma_{k-1} + E(\epsilon_t X_{t-k})$$
根据$\epsilon_t$独立于$X_{t-k}$，且$X_t$均值为0，所以$E(\epsilon_t X_{t-k}) = 0$，即：
$$\gamma_k = \rho \gamma_{k-1}, k = 1, 2, 3, …$$
将k带入求得$\gamma_k$：
$$\gamma_1 = \rho \gamma_0 = \rho \dfrac{\sigma_e^2}{1-\rho^2}$$
$$\gamma_2 = \rho \gamma_1 = \rho^2 \dfrac{\sigma_e^2}{1-\rho^2}$$
$$…$$
$$\gamma_k = \rho \gamma_{k-1} = \rho^{k} \dfrac{\sigma_e^2}{1-\rho^2}$$
即：
$$\rho^k = \dfrac{\gamma_k}{\gamma_0}$$
自相关系数只与间隔(k)有关，因此是平稳(stationary)的。
